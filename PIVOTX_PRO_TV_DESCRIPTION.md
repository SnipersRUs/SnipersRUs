## 📊 KEY FEATURES

### 1. Dynamic Pivot Detection
- Uses ATR to calculate pivot strength dynamically
- Adapts to different timeframes automatically
- More accurate than fixed pivot strength

### 2. Market Structure Analysis
- Detects CHoCH (Change of Character) points
- Tracks bullish/bearish structure
- Visual markers for structure breaks

### 3. Pivot Zones
- ATR-based zone sizing
- Extendable zones showing future interaction areas
- Color-coded (green for support, red for resistance)

### 4. Fibonacci Tool
- Auto-draws between major pivots
- Shows only key levels (0.236, 0.382, 0.5, 0.618, 0.786)
- Golden Pocket (0.618) highlighted
- Off by default (enable when needed)

### 5. Multi-Timeframe Filter
- Filter pivots by higher timeframe trend
- Configurable HTF (default: 4H)
- Optional feature

### 6. Comprehensive Alerts
- Pivot High/Low detection
- Major pivot zones
- Market structure shifts (CHoCH)
- Exhaustion signals
- Fibonacci level touches
- All individually controllable

## ⚙️ QUICK SETUP

### For Day Traders (3m-15m)
1. Enable ATR Confirmation
2. Enable HTF Trend Filter (set to 1H or 4H)
3. Enable Pivot Zones
4. Configure alerts for your needs

### For Swing Traders (4H-Daily)
1. Disable ATR Confirmation (or use loose settings)
2. Enable HTF Trend Filter (set to Daily or Weekly)
3. Enable Fibonacci levels
4. Focus on major zones and CHoCH signals

### For Scalpers (3m-5m)
1. Use tighter ATR confirmation
2. Focus on pivot zones
3. Monitor exhaustion signals
4. Use CHoCH for structure breaks

## 📈 SETTINGS GUIDE

### Pivot Detection
- **ATR Multiplier**: Higher = fewer, more significant pivots (default: 0.5)
- **Volume Threshold**: Minimum volume spike (default: 1.5x)

### Filters
- **HTF Trend Filter**: Filter by higher timeframe trend
- **ATR Confirmation**: Require price confirmation (recommended: ON)

### Visuals
- **Pivot Zones**: Show support/resistance zones
- **Pivot Labels**: Show PL/PH labels
- **Fibonacci**: Auto-draw retracements (OFF by default)

### Alerts
- Individual toggles for each alert type
- All enabled by default (except Fibonacci touches)

## 💡 PRO TIPS

1. Start with default settings and adjust gradually

3. Enable ATR confirmation to reduce false signals
4. Focus on major zones, not every pivot
5. Combine with price action for best results
6. Golden Pocket (0.618) is statistically the most reliable level
7. CHoCH signals indicate major trend changes
