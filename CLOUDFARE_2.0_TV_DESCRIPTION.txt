7. TREND SCORING SYSTEM (0-10 SCALE)
Multi-factor classification evaluating price vs MAs, MA relationships, momentum, and patterns. Provides clear trend context for all signals.

Optional Asian, New York, and Current Session boxes tracking session high/low ranges with transparent visualization. Current Session box (yellow) updates in real-time showing today's trading range.
