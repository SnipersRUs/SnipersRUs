# Sniper Mini VWAP - TradingView Description

## Overview
Enhanced VWAP indicator with multi-session ORB (Opening Range Breakout) strategy for crypto trading.

## What's New (Updated Version)

### ✨ Multi-Session ORB System
- **Automatic ORB calculation** for all 4 major forex sessions (Sydney, Tokyo, London, New York)
- **24-hour ORB boxes** - Each session's opening range extends for a full 24 hours for better visibility
- **4-day history** - Shows ORB boxes from the last 4 days (configurable)
- **Session-specific colors**:
  - Sydney: Cyan border
  - Tokyo (Asia): Blue border
  - London: Yellow border
  - New York: Orange border

### 📊 Enhanced VWAP Display
- **Thin cross-style lines** for all VWAP levels (cleaner chart)
- **Updated colors**: Neon Green (1H), Neon Yellow (4H), Magenta (8H), Red (Daily)

### 🎯 Simplified Signal System
- **Buy signals**: Green circle when price breaks above any ORB high
- **Sell signals**: Purple circle when price breaks below any ORB low
- Only major breakout signals - no clutter, just the important moves
- Signals track all active ORBs across all sessions

### ⚙️ Key Features
- **Volatility adjustment** for crypto markets (adapts to different exchanges/assets)
- **Clean visual design** - transparent ORB boxes with dashed borders
- **No text labels** - just simple dot signals for clarity
- **Automatic session detection** - no manual time setting needed

## Perfect For
- Crypto traders looking for opening range breakout opportunities
- Multi-timeframe analysis with clean VWAP levels
- Traders who want to see when price returns to ORB ranges for potential trades
